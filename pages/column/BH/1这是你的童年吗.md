## 这是你的童年吗 #1
兔斯基，是由中国传媒大学动画系04级毕业生 **王卯卯（MOMO）** 创作的一套动画表情形象。后被时代华纳公司旗下公司特纳中国收购，成为其旗下核心IP。就是那只耳朵细细长长、脸长得一副嫌弃的样子、转动着两根面条般的手臂做着搞笑动作的兔子。\
![兔斯基](/images/column/BH/tsj.webp "兔斯基")

阿狸，当红原创动漫形象，性格可爱、温暖、孩子气，一直相信童话。于2006年由**Hans**创作，从网络与杂志连载开始，发行童话绘本和动画短片。\
![阿狸](/images/column/BH/ali.webp "阿狸")

《小破孩》是2002年发行的系列动画，具有浓郁中国特色和时尚元素。原作者**田易新**(笔名拾荒，中国，汉族，中国动画制作人。河南洛阳人。)\
\
![小破孩](/images/column/BH/xph.webp "小破孩")\
\
以下是东方竹子(原名毛竹)女士的文章

### 小破孩的原创————天下最有灵气的大男孩拾荒
我说：你和小破孩真的挺神似的只是你的额头上少了小破孩的“爱情大包”。我细细看拾荒的额头，发现那上面不仅没有“爱情大包”反而有一点微微凹陷。不知道是不是“爱情凹陷”。那可是神秘的“爱情凹陷”，是神秘的“爱情天坑”吗我的脑海中闪过浙江境内神秘的接天地灵气雾气迷蒙的天坑。\
\
和拾荒、田平易哥俩和投资公司郑总会面后在上海的街头合影留念。\
\
第一次见到小破孩和小丫——那时还不知道他们的名字，只是知道是一个大脑袋、穿小裤叉的小男娃娃和一个穿红兜肚、梳两个朝天小辫子的小女娃娃，更不知道小男娃娃的小裤叉是一个小宝库从中可掏出各种各样的好玩东西，更不知道小破孩的大额头上的那个似寿星老人的额头上的红桃儿是为了给小丫拽月亮，被月亮砸出的一个“爱情大包”。\
\
那是青海朋友qh通过电子邮箱给我发来的一个十一贺卡，里面有十多张动漫，其中有三幅是关于小破孩和小丫的动画。一幅是两个小娃娃牵手走在风雨中－－那小男娃娃的两个“对号眉毛”下有圆圆的眼睛，那嘴唇居然是斜立着的，且走一步嘴一歪，走一步嘴一歪；那小女娃娃的眉毛居然跑到发际线那去了，那小腿儿是内拐子一般，可爱可笑。小人物拉着小媳妇生存的艰难被那么惟妙惟肖幽默有趣地表达出来；一幅是那个小女娃娃在后面抚摸那个小男娃娃，那个被疼爱的小男娃娃两只小手向后乍着，脸上幸福的表情真是让人心花绽放；一幅是两个小娃娃做出泰坦尼克号船头上的抒情状，陶醉在迎面扑来的风浪中。居然一下子喜欢上了这两个带有中国特色的动漫小娃娃。\
\
一次一次看，心里一次一次地欢呼起来：这两个小娃娃真是太可爱了！那时根本不知道这两个小娃娃的创作者是谁，只是觉得这两个小动漫小娃娃真是太心疼了！啥叫“一见钟情”。这种，也可以叫！\
\
看不够！怎么办？于是就把这两个可爱的心疼的小娃娃收在我的图片收藏夹中。闲时打开看，这两个小娃娃居然还可以动呢。他们或向我走来，或在抚摸中陶醉，或在沐浴船头的风浪，那小样子真是要多可爱有多可爱，要多心疼有多心疼。我的心里真是说不出的惊喜！于是，忍不住叫朋友们来看，来看这两个稀奇的小娃娃。\
\
从那以后，在不知道的瞬间，脑海中会闪出这两个稀奇的动漫小娃娃。在我见到他俩的那一瞬，他俩已经毫不讲理地闯入我的生命，且再也不肯离去，他们在我的生命中浪漫地恩爱着，好玩地生活着。而这种霸占，居然带给我那么一种由衷的惬意。\
\
有一天，向伴儿推荐这两个神奇的动漫小娃娃时，忽然冒出一个想法：若是能找到这个两个动漫小娃娃的创作者该多好！这想法一闪而过，如同幽蓝的宇宙中有美丽的流星雨从天空掠过。\
\
有一天，在新浪的主页上我又看到了这个两个动漫小娃娃，看到下面有三个字“小破孩”。我心一动，原来这个动漫小男娃娃名叫小破孩。我心一喜，既然知道了他的名字，搜一下，不就可看到更多的小破孩了吗？我一搜，果然搜出好多的小破孩。这可真是“千呼万唤始出来，犹抱琵琶半遮面”！让我惊喜的是，这个创作者的水平是高超而稳定的，因为每一幅有关小破孩的动漫都是那么有灵气那么令人叫绝。就连小破孩玩木马，小破孩被揪耳朵，小破孩打呼噜、小破孩成杀手，小破孩变后翌射月，小破娃拿大碗喝酒、小破孩求婚、小破孩抱大奶瓶、小破孩洗脸，小破孩戴口罩、小破孩不舍得给小娃娃喂奶，等等在生活中极普通的细节都被表现的充满了灵气。特别是小破孩那标志性的走路时向下歪斜的小嘴，有时居然可变成惊叹号下面的小水点，有时居然可变成问号下面的小亮点，有时可以变成冒号两点中的一点。突现了小破孩的好奇心的同时生动表现了小破孩经历的喜怒哀乐，幼拙稚气中妙趣横生。更有那大脑袋上的大包、有些内拐的小脚丫子、微腼的小肚子、有些罗圈的小胖腿、踌躇满志的走路姿式、仰头女子低头汉的小德性、傻乎乎却又大智若愚蠢的皱鼻子，都是是中国味儿的，都是那么可爱那么心疼那么有趣那么幽默。就连小破孩有关的：弃婴的嘴居然花瓣儿一般，六一儿童节那一帮小人人都是那么可爱有灵气有新意。我试着全体收藏，可是总是不能如愿。\
\
我回到家，如同发现了新大陆一般，招呼我的伴儿过来看我搜出的小破孩。也就是在这一瞬，我忽然想：既然我可搜到小破孩动漫，为何不能搜一下小破孩创作者呢？我的记者职业病又犯了。而我建立的“东方竹子国际互联网”的初衷正是关注这个时代真正优秀的幕后人，这个时代真正的创造者，这个时代真正的决策者。一句话，那就是实实在在影响这个时代的创新者创造者决策者，而不仅是歌手演员这些浮在生活表面作秀的“木偶”。也就是说，关注的不仅仅是这个时代光亮鲜丽的外表，而是其内核。我果然地在网上搜到了小破孩的作者笔名拾荒的田易新。\
\
我打电话到上海，居然很顺利地就接通了田易新的电话。\
\
田易新在电话中告诉我：你看到的只是一些画面，是小破孩动漫短片中的一些瞬间动作，你可在我们的网站中看到许多的小破孩动漫短片。拾荒说，小破孩五年了。\
\
我问：这么好的创意是怎么出来的？拾荒说：中秋节前两天，我收到一位朋友通过网络发来的中秋动画贺卡。可是我在网上想找个动画图象回个贺卡时，却找不到一个合适的动画形象。于是，我只好孤注一掷，自己创造一个!我的脑海中突然冒出我们洛阳村里庄里满地跑的只着小裤头的小屁孩，家家宠爱的只穿小兜肚的小丫头，找人到灵感的我连续忙碌了两天两夜，创作了那个3分钟的动画短片《中秋背媳妇》――后来我知道就是那个看起来有些粗糙、两个小人脸鼻模糊、两个小人人还不太可爱、两个的爱情只隔一水而不是后来“包”中那些惊险的那一个动画短片。那个动画短片中，小破孩听到一小丫的哭声，小破孩呼哧呼哧游寻到在海水那一方。小丫在那哇哇大哭呢。小破孩，用他的的小裤头中掏出糖葫芦，不行！玫瑰花，不行！一吊钱，还是哄不好。小丫对着月亮继续号啕。小破孩像是明白了小丫的心思，解下小裤衩，一个甩手，用裤衩拉下了中秋的月亮。小丫不哭了，把它当月饼啃。可是，当小破孩转身要离去时，小丫又开始张大嘴哭。于是，小破孩一把夹起小丫，带着她游回了家。背景音乐是浪漫的《梁祝》，字幕上打出一行字：给没有家的人。\
\
那个动画短片无疑是珍贵无比的原版，是小破孩和小丫已见雏形的胎胚。这一对可爱的中国娃娃呱呱坠地，男娃我给他起名叫小屁孩－－我们洛阳人把小男孩叫小屁孩――也就是小破孩，女娃我给起名叫小丫。在这部短片里，我让“小破孩”、“小丫”两个婴儿，在中国音乐《梁祝》里演绎中国式的爱情故事。这个短片做完后，已经是中秋节的凌晨，差点过时了！我给朋友回了贺卡后，还在国内一些门户网站上公布了此片。之后，我困死了，整整三十多个小时在激情亢奋中“生产”，终于产出了这两个双胞胎小娃娃。我倒头就睡，一觉醒到月亮升起，我习惯性地到网上看看，哇！铺天盖地的小破孩，铺天盖地的网友来信，网上反应之热烈，让我震惊——短短一天，《中秋背媳妇》在各大网站的点击总数居然超过1百万次，上万名网友在论坛和我的个人网页上留言，对这个动画短片进行了热烈的评价。\
\
这对我是极大的鼓舞。我忽然感觉到就是在这两天，我孕育的渴盼的期待的什么终于诞生了。我迷惘了这么多年，我浪迹了这么多年，我痛苦了这么多年，我阵痛了这么多年，我迷失了这么多年，我觅寻了这么多年，我从这个地方到那个地方，我从这个制片厂到那个制片厂，原来就是为了他俩。我有一种阵痛后的快感。\
\
我立即静下心来，对“小破孩”、“小丫”的形象进行再加工，创作了《金瓶梅》、《景阳冈》、《七种武器》、《佐罗系列》等“小破孩”系列动画，使这两个形象更加完善。\
\
每一部短片做出后，我就把它们挂在网上，于是更多的网站出现了小破孩和小丫。小破孩受到那么多人的喜欢。让我深感慰藉。\
\
当然，网上增添的不仅有对小破孩的赞杨惊叹，还有指责怒骂。\
\
我后来看到拾荒的创作的《倒拔垂杨柳》，那垂杨柳像一个孔夫子，猜想可能和小破孩受到指责的这一段的经历有关。这里面一定有小破孩既要继承传统有想反叛传统的心路历程有关。\
\
我问拾荒怎么能创造出这么可爱的带有中国民族特色的动漫人物。\
\
拾荒谈到了洛阳是经历过十几个朝代、孕育过上百个皇帝的都城，以至于别说城里就是村里庄里施工挖地都要上报审批，随处都可挖出珍贵文物，随处都会挖坏珍贵文物。拾荒说正是洛阳这片土地孕育了他且给了他无限的灵感。洛阳随便一个门脸随便一个地摊上都能找到特别好的创作。真可以说灵感遍地珍贵四溢。拾荒说他的创作的立足就是带有洛阳特点的中国文化。这或许就是所谓的“草根文化”。\
\
我想起那一次去河南济南洛阳等地旅游，导游讲的大意思是：这个湖下是某个朝代的湖，某个朝代的湖下又是某个朝代的湖…………这种湖下出现的层层叠叠的"朝代"带给自己特别的震惊。而那次去在涿州人和刘备、张飞三结义的关羽留在洛阳附近的身冢－－张飞的头在三峡的张飞庙，身在何处？关羽的身在洛阳，头在何处？桃园三结义的三人居然有两人是身首异处，这更是带给我的朝代乱世的交错感。\
\
原来拾荒立足的是东方文化，小破孩和小丫是被中国这片古老神奇的土地上注入了神气灵气，才真正地活了。\
\
我说我的写作也是立足东方文化，故而我的笔名叫东方竹子。\
\
在电话中我们有一种知遇之感。\
\
我想说，创作者最最难能可贵的便是：能够认识自己，能够认识乡土文化的氛量，能够认识自己、自己的省、自己的民族、自己的国家和其它的比有哪些不同。这种认识通常是在周游世界了解了外面的世界之后，才能明白的。就如我在青海时十分自卑，可是真正周游中国见到过各地名胜、见识过各大家名家且得到他们的普遍赏识之后才敢带着突破性地意识到青海文化在中国文化中的分量，认识到大巴山故事的与众不同。\
\
而我设想，可能是某个夏天，拾荒在外面飘泊地困了，流浪累了，求索倦了，唱着“归来吧！归来吧，浪迹天涯的游子。我曾经毫情万能丈归来却空空行囊，那故乡的风，那故乡的云为我抚平伤痕”，独自徜徉在洛阳的街头时，忽然发现村庄里河沟边、山坡上随地乱跑的只着一个小裤头的小屁孩，和只穿一个兜肚的小女娃，十分可爱，拾荒痴望着这些小屁孩小丫头，不由想起了自己小时候，想起小时候的种种有趣种种好玩。这个感觉被拾荒珍藏在心底。以至那两天，这个灵感忽现，促使拾荒创作出了这一对可爱的东方特色的动漫人物。\
\
拾荒评价自己：也算是飙人类吧，因为中国动漫没有走过的路，我们都勇敢地去尝试了一遍，虽然那叶方舟还没有抵达幸福的终点，但它已经行至河间，最重要的是它已经看准了方向，知道彼岸在哪里。\
\
我说：你的小破孩缺少的仅仅是好的独独属于小破孩的动漫剧本和令人叫绝的独独属于小破孩的音乐。从动漫的角度你已经达到了一个很高的境界。\
\
我的自认为我的眼光是很准的。这也是这么多年实践证明过的。我认识的许多人，我认识他们时他们并不怎么出名。可是认识他们并预言他们的成功后，他们居然很快就非常成名。这也是奇迹呢！带给我发现的乐趣!\
\
你立足于东方文化。但这种立足仅限于诠释古典的中国文化、再现现代的中国文化、甚至利用外国文化来诠释中国文化，虽然诠释得很有新意、很有灵气、很有创新，但还不是一种根本意义上创新和突破。也就是形式达到了，内容上也达到的，但是还没有真正地达到。我说的，也就是非小破孩莫属的新东西。否则你的小破孩就无法获得真正的生命，也无法获得恒久的生命力。\
\
拾荒十分同意我的观点。\
\
但是我心里明白，小破孩已经出名，想找原创好本子或是原创好歌曲都当是很容易的。只要拾荒团队有一种海纳百川的胸怀气度和一些慧眼识英的当代才俊。别说歌手演员，就说作家：当年的琼瑶，歌词一写出来，便有无数的人来为她谱曲，无数的人来为她制作，无数的人来为她拍摄。真可谓“千万宠爱于一身”，能不出好歌吧？琼瑶只是一种虚心态席就给琼瑶带来了可能流芳千古的歌。\
\
这是一个过程，小破孩的发展的前期，当经历的是三个飞跃。经历了第一个飞跃，也是最难的五年，那就是打出品牌；而第二个飞跃，这第二个五年，是最辛苦的五年，当是做成产业；第三个飞跃，这五年，才当是注入真正的全新的东方文化内涵－－这是一个最痛苦的过程，是要经历蜕皮脱胎的，是最难的一个飞跃。从真正意义上来说只有实现了这三个飞跃小破孩才能真正横空出世。\
\
小破孩，只要能保证上亿的“宠爱”于一身，早晚都会出来独属小破孩的更好短片和独属小破孩的更好歌曲。这一点只要拾荒公司做稳做实，发展的同时能不断吐故纳新、创新突破，采取虚心的态度接纳，用自己的眼光寻找，就会水到渠成。\
\
特别是小破孩那标志性的走路时向下歪斜的小嘴，有时居然可变成惊叹号下面的小水点，有时居然可变成问号下面的小亮点，突现了表现小破孩的好奇心的同时生动表现了小破孩经历的喜怒哀乐，幼拙稚拙中妙趣横生。更有那有些内拐的小脚丫子。我试着全体收藏，可是总是不能如愿。\
\
我说：你和小破孩真的挺神似的只是你的额头上少了小破孩的“爱情大包”。我细细看拾荒的额头，发现那上面不仅没有“爱情大包”反而有一点微微凹陷。不知道是不是“爱情凹陷”。那可是神秘的“爱情凹陷”，是神秘的“爱情天坑”吗？我的脑海中闪过浙江境内神秘的接天地灵气雾气迷蒙的天坑。\
\
我说从网上看到你被称作“中国十大闪客”，说你们三个洛阳人投资50万元建立了拾荒公司。\
\
拾荒说：那时的我们哪儿有五十万元！动漫也不挣钱，挂在网上亿人看都是免费的。当时是二万元建立了拾荒公司。\
\
－－我后来看了拾荒导演的佐罗系列招聘，想起拾荒的这段话，推想拾荒可能用小破孩之嘴表达了自己求索中的艰难：“哎，做侠客和做闪客一样，看起来挺风光的。其实呢？饥一顿饱一顿的。好几天没吃一顿饱饭了。下个月的房租还没有着落呢。虽然我武功高，但也不能去抢劫呀！我当去找一份工作。”小破孩经历的这些辛苦辛酸也只有拾荒的心里最知道。而招聘还有更耐人寻味的细节：比武，三诫一顿乱打，打倒木人一片，获得喝彩一片，只剩下一个木人让小破孩显身手。小破孩用尽全力拼杀最后的木人，可是旁观者却看不到木人有何动静，只有等他们都走了，那木人才一块一块很"精致"很"别致"地倒下。可怜的小破孩指着，想让他们回来看，可是他们没有人回头看。而武功高小破孩居然只因吃得少被聘用。我想这些细节都有拾荒创作的坷坎有微妙的联系。当然只可意会不可言表。这也是拾荒动漫的微妙之处。\
\
在投资额做为商业机密，投资内幕作为商业隐私的滚滚红尘中，我从拾荒的这一句话中一下子就感觉到了拾荒做为真正艺术家的真纯、可敬和可爱，还有重要的一点就是可交。\
\
我问拾荒：是哪个学校毕业的？怎么能做出那么好的动漫？\
\
拾荒说：我考大学挺不顺利的，先是上了洛阳一所艺术学校，好在老师是一位在洛阳很有名气很牛B的人物，他的学生在当地都很有出息很出名。在这位牛B老师的指导下，我的绘画突飞猛进，第二年，我就考上了洛阳某大学，可是大学我只读了一年我就感觉这样学下去没有前途，我主动辍了。我先后在上海美术电影制片厂、中央电视台下属动画公司从事动画制作工作。国外动画的疯狂入侵和中国动画事业的江河日下，让我陷入痛苦的思考。在没有画小破孩之前，我的日子过得挺滋润的。那种痛苦是一种精神上的痛苦。动画界都知道的秘密，中国美术劳动力价格便宜，因此成为了美国和日本、韩国等动画先进国家的加工商。当我们的温饱成了问题，只好把胃交给了西方。其间，像我这样的资深人士一个月挣个一两万是不成问题的，但是却没有自己的思想，这正是我痛苦的根源。\
\
我不乐意一直这么干下去，我的心里装着一个可爱的、固执的理想，就算成不了查尔斯·舒尔茨、埃·奥·卜劳恩或者宫崎骏，如果可以画自己想画的东西，恐怕就不会那么郁闷。\
\
我2000年从央视辞职，带着照相机周游全国，一方面释放的我痛苦，一方面享受心灵的自由，不自觉地在苦闷中寻找灵感。2001年，我回到洛阳，无意中发现了通过FLASH进行网络动画创作的全新方向，并创作了《庸人自扰》、《龙门客栈》、《黑色故事》、《二蛋传奇》等FLASH作品。直到2002年小破孩出世，我都处在一种迷茫的觅寻状态。那也是一种阵痛状态。从我生命中横空出世的小破孩、小丫让我有了方向，也让我明白了这些年我一直在苦苦觅寻的到底是什么。\
\
拾荒忽然问：你的报社在什么位置。\
\
拾荒说：我以前住的地方离你很近。我在那住了好几年。几年前，我才到上海。\
\
“中央电视台不是在军博那边？”“中央台视台下属动画公司在北京电影制片厂。”“那真是太近了！”。\
\
我们都感叹：原来我们曾离得那么近。可是相近咫尺时我们却不相识。\
\
我说：早结认识你就好了，我们离的可就远了，见一次面也不容易呢。\
\
拾荒说：我经常地去北京，几天前刚去过呢。我再去北京会去看你。欢迎你来上海到我们公司来。\
\
我把我的关于奥运的一个灵感告诉拾荒，激起了拾荒的兴趣。我们约定尽早见面。电话中，我们都表达了希望能早些面谈的愿望。\
\
根据拾荒提供的线索，整整几天，我都在网上搜索，寻找了许许多多关于小破孩的动漫短片、壁纸彩信。那两个小娃娃这下子可是真的活了，真是太可爱太有趣了。那个小破孩鼻子居然是“抽抽”着的，脸上还有小脏猫似的右三道左三道，那个小破孩挠鼻子挠身子的居然带着傻傻的坏笑；那个小丫哭得时候嘴张的居然可以比身子还大，里面一个扁桃体如红月牙儿一般，那小丫的眼泪居然可以连续“镖”出，这样看起来小丫就像“泪丫”一如我小时喜欢抓的一种虫儿:花大姐。更好玩的是小破孩的小裤头居然神里，每次哄小丫小破孩最拿手的只有三招：糖葫芦、玫瑰花、一吊钱——这是拾荒创造性的用活了民间流传的一幅小女孩看小男孩子秘密的的画，且叫人惊奇那里还掏出骗小女孩的三件礼物。后面还能变成一个想要啥就有啥的聚宝裤头呢。\
\
看完了，回忆那两个小人人，我的脸上荡漾着笑，心里花开着笑。拾荒创作出的两个小娃娃活了，打倒了拾荒，正如拾荒的作品《夜半惊魂》。每一个动漫我都看了好几遍仍意犹末尽，还想看。觉得拾荒的作品还不够多，不甘心地搜呀搜。更加难能可贵的是贯穿始终的小破孩子身上的那种好玩幽默灵气，那真的是一种中国小人物的可敬。小破孩的普及性和亲和力可能也源于此。每一个普通的男人都从小破孩的身上找到了自己的影子，每一个普通的中国男人都看到自己生存状态被一种轻松的方式诠释出来。\
\
心里实在是充了了惊异，这个神奇的大男孩拾荒！《弃婴》《牛皮癣》《西门庆》《武松打虎》《佐罗系列》等都十分精彩，就连庆祝六一、庆祝圣诞、庆祝三八妇女节、庆祝愚人节、防非典、防火这样的老套生硬的题材都被拾荒导的那般有灵气，甚至令人叫绝。\
\
而精彩的细节更是多。就连小破孩的脚印--那沙漠中两串忽隐忽现的枫叶，小破孩的泳态--那臭德行样子，那茫茫海水中隐现的“水梯子”，都那么有神气。\
\
更为新鲜的是，拾荒在动漫人物还能申扬的中华民族的正气呢！那么自然亲切随和感人幽默有趣的一种方式，什么反对倒卖妇女儿童，什么反对乱贴广告，什么乱扔口香糖，什么救火等等，而这种动漫在社会上普及取得的社会效益是正规教育不能企及的。\
\
回过头再看其它的中国动漫片子，忽觉得都黯然失色，心里唯留小破孩。更为惊喜的是拾荒是创新是突破，是带有中国特色的动漫。他的立足和我一样都是立足东方文化。我真想感谢上帝创造了小拾荒。\
\
这世上的事情真是神奇！就在我叹息不知道何时何月何年才能见到神奇的大男孩拾荒的时候，教育部作销售经理培训的女友张俊玲打来电话，说她要去上海做业务，希望我能陪她去一趟上海。且机票全由她包了。我推辞不过，加上对这个神秘美女的生活有一种好奇，于是一同飞往上海。\
\
我和张俊玲下塌在华东理工大学附近的惠谷港弯。第二天，张俊玲去开始业务，我便按拾荒提供的地址找到锦江之星宾馆，我打电话拾荒请他派个人下来接我。\
\
不一会儿，听到有人喊我的名字，我回望，原来是拾荒亲自来接我。我心里好高兴。\
\
拾荒身高大约有一米七几，穿一身休闲服，比电脑上看到的他更显潇洒、俊朗、随和、睿智。只是和那些或是长发飘逸、或是扎个马尾的艺术家们不同，拾荒是典型的“光光头，卖酱油”。我问拾荒那光光头是天生不长毛还是剃的？拾荒摸着自己的光光头说：我的头两周剃一次，很多年了。拾荒走在我前面时，我发现拾荒后脑上有三道纹，而拾荒创作的《夜半惊魂》中的拾荒后脑也有这三道纹。像与不像，拾荒创作时抓住的总是神似而不是形似。就如小破孩和拾荒。拾荒本就是一个上帝创造出精彩的大男孩！\
\
我在网上看到接受记者采访的拾荒时，就有这个发现，拾荒和小破孩很神相。这样近距离看拾荒，感觉拾荒和小破孩更神似。难道小破孩子的原型就是拾荒自己？这个发现使得小破孩变得更加亲近，更加感人。从拾荒中提练出好玩的幽默的风趣的小破孩，真的很神奇呢！\
\
我说：我发现了一个奇迹，你和破孩子真的很神似！我强调那只是一种神似！\
\
这个发现虽然普通，但是在别人还没有发现，或者在别人发现了还没有说出，或是别人发现了还没有写出时，那便是我毛竹的一个伟大的发现，可以当毛竹“专利”呢。写到这毛竹自己都笑了。\
\
拾荒也不回避，说：画家越画就是越像自己。\
\
我想起法国卢浮宫里的名画《蒙娜丽莎》。有新说法，说蒙娜丽莎之所以笑得那样安静那样神秘，不是因为她的生命中孕育着新的生命，也不是展示她的温馨妩媚，更不展示女性生命丰蕴美丽，也不是暗示她和作者达芬奇间的微妙关系，而是蒙娜丽莎的长相就是达芬奇本人。还有人做了一个换像实验，果然很像。神秘的面纱一旦揭开，果然是石破天惊。更让人惊叹的是，蒙娜丽莎是女的，达芬奇是男的。而拾荒和小破孩的神似，毕竟没有性别的阻绕，气场是对的，让人更容易更乐于接受。\
\
而对于小破孩的喜欢者可能最大的慰藉就是小破孩神似拾荒了，因为这会使我们对小破孩的兴趣有延续持久的动力，因为这个形象有一个活生生的生命来支撑，源泉是活的，源泉是动的，是有血有肉的，这样的小破孩当然会更有魅力。\
\
我说：你怎么能想到把小破孩的嘴有时弄着歪斜的？很有灵气呢。拾荒说，这很正常，你观察过婴儿吗？因为他们太小，嘴不小心就歪了。\
\
我心一动，难怪田易新自起笔名拾荒，他真的能在现实生活中拾出这么多的精彩。更让人欣慰的是这么精彩的一笔，在拾荒的心里居然是很平常。而拾荒的许许多的精彩就是这样从现实生活的大“拉圾堆”中拾来的吗？\
\
我浮想连篇。这才叫真正有创造力的艺术家。而许多艺术家习惯在别人的作品中找“灵感”，有的甚至把别人的作品改头换面。动漫作品是，文学作品更是－－比如作家苏童的《妻妾成群》后改成电影《大红灯笼高高挂》给人的感觉是来自《红楼梦》却写不过红楼梦，亮点是写出的男权主义统治的中国某些成功男人在女人问题上的自私和阴险和毒辣，唯二的亮点是写出了这个环境下生出的儿子害怕死了勾心斗角的美丽女人而成了一个怪异同性恋；比如著名作家贾平凹，起《废都》这个书名真是起绝了，写男人的内心世界真是写绝了，可是平凹笔下的女人却写不过金陵笑笑生《金瓶梅》中的女人，更写不过曹雪芹《红楼梦》中的女人，根本没有一个立住的女人。而小破孩的出世从动漫和细节的角度却是完全的创新，完全的突破。真可谓“前不见古人，后不见来者”！这种精彩从动漫的角度才叫全方位的精彩。难道唯有拾荒能够完成这种彻底的淋漓尽致的新创突破？就是因为拾荒在现实生活的“大垃圾”堆中翻来覆去寻找灵点？\
\
我和拾荒边走边交谈，发现这个大男孩拾荒智慧、稳重、大气、内向、执拗、虽然有些不苟言笑，但小破孩的身上的一切优缺点一切缺点拾荒仿佛都有，所不同的只是拾荒把小破孩身上的幼拙天真幽默玩皮和玩都紧紧收藏在生命深外，显山不显水，似乎是要做出一种大人的深沉。那是一个成熟和长大了的小破孩！\
\
拾荒带着我绕到锦江之星宾饭后面，进入一个大铁门，又绕了一个半圈才到了他的公司。塔楼23层的一个大房间，客厅里面有十几台电脑前坐着些年轻的设计者们，迎面两个小房间，右手的小房间里有十来台电脑前坐了些年轻设计者们，左边一间就是拾荒的办公室。这办公室的三间屋中都贴了许多小破孩的画框，小破孩情侣装的广告等。\
\
在拾荒的办公室，拾荒谈到拾荒公司正想让动漫小破孩成为一种中国民族特色的产业。谈到已经和许多的公司达成了小破孩商标授权。比如儿童服装等。奥运会的“伴嘴”瓜子也印上了小破孩形象。谈到了成功推出的几件小破孩商品。提到有半屋子里装的都是失败的小破孩玩具――要想把小破孩从屏幕上搬下来从平面形象变为立体形象可是不容易呢，拾荒公司正在进行着新一轮的艰苦的探索和尝试。拾荒特意谈到拾荒公司遇到的困难和取得的成绩。临走，拾荒把我带到楼上，让我选一对情侣小破孩T 恤。我选了一黑一红两件小破孩T恤。\
\
我看到有一对毛绒绒的小破孩和小丫，连说这个放在车后面真的不错，可是拾荒说这是样品还不太满意，故而不能走向市场。\
\
下楼拾荒又送我一套南京音像出版社出的纪念小破孩五周岁的MTV光盘和图书。我问出版方式。拾荒说：南京人民出版社表态：宁肯亏也愿意出！我在心里佩服南京人民出版社的某位决策者。因为这可是我们的民族动漫品牌，是中国第一个呢！等到以后小破孩有无数的乌合之众时，这种亏着也要做的人或公司，才是真正慧眼识英且应当写入中国动漫发展史册的。\
\
拾荒送我走时说希望我第二天能再来，因为有许多事情需要和我商量。\
\
于是，第二天中午，我推掉和张俊玲的约定，再去看拾荒。出发前我给拾荒发一个信息，告诉我已经上路。\
\
路上，我的手机响了，是我上海的亲友打来的。我蹲在路边打电话一打就是一个多小时。\
\
这时我的手机进来了一个信息，一看是拾荒发来的：“是不是迷路了？”我笑了，这可爱的小破孩式问候方式终于出来了。虽然拾荒是大人了，且正按大人的要求为人处事，且按大人的要求运作拾荒公司，可是不小心暴露出来骨子里还是一个小破孩，或者说是“童心永驻”吧！或者是为小破孩保持住的这一份天真吧!或许是这一份天真由小破孩不经意表达出来吧！不然怎么唯独他可能创新出那么可爱的动漫人物呢？\
\
这一句话的感觉和他前几天的电话的感觉，昨天所有的话的感觉都不一样。可能是通过昨天的沟通有了一种了解，朋友关系更进了一步，人也放松了，这种小破孩式问候方式也就出来了。\
\
这样想，迷路这件事情也变得美丽起来。而拾荒真是猜对了，我真的是一边打电话一边迷路呢！只是电话没有打完，还没来及问路。\
\
我再次进到拾荒的办公室。\
\
拾荒知道我热了渴了，递给我半瓶鲜橙汁，说：这是你的！我定眼一看，可不是？那真是昨天拾荒专门为我在超市购买的。买前拾荒还特意问我喜欢喝哪种。我昨天喝了一半的那瓶。昨天，临走时我感觉拾荒望了望那半瓶，知道他是想让我带上路上喝，但是没好意思开口。却是留着让我再来时喝。他难道这样的留着就是期待着我今天再来吗？为了这个细心的大男孩拾荒，我笑了，笑得好由衷。而拾荒动画的出奇制胜，也正是因为他的动漫中有太多太好的细节。\
\
拾荒问我是不是坐地铁来的。我说不是，但是坐过上海的地铁。拾荒说，有没有看到地铁公司播出的小破孩短片？拾荒特别提到了新兴媒体覆盖全国多座城市的公交电视——“巴士在线”、利用出租车电视播出的“上海东方明珠城市电视”、在广东省手机电视上播出的“广州雅信手机电视”、在杭州公交车上播出的“杭州视播公交媒体”等，均播出了“小破孩”系列动漫作品。播出后反响强烈。\
\
拾荒给我看一张纸，上面画有一长溜子小破孩，那一长溜子小破孩正在做一个后跳的动作。拾荒说：你看看，小破孩往后跳一下，我们就需要画这么多张动画，真的很不容易呢。每一部作品的推出都很不容易呢，观众看着几分钟，我们这么多人需要费尽几个月的心血。\
\
我说：可是在我们观众的心里，总觉得小破孩都五年了，可作品还不够多。因为好看我们没看够。我们想看更多的。我们在期待着下一部。\
\
拾荒拿出他设计的新作品《王二小》的草画，以享竹子。我一张一张看，每一张都画的很有功底，光看手绘的画就足见力度，王二小的故事一张一张展开，且那些灵点已经绘出，带着成熟的手法和遒劲的笔力和特别的新意，光画面就感动得我的眼泪快出来了。\
\
拾荒说，这里面小破孩子和小丫不是情侣而是儿童团的小伙伴。我说这也适合，在我们心里他们当是青梅竹马的一对。\
\
拾荒说：这一次的《王二小》动漫短片会配合大家都熟悉的歌曲《王二小》。最后的结局也不是那么悲惨，观众只要点击一下，给王二小喂一瓶奶，就可救活王二小。我觉得很有创意。\
\
昨天我曾向拾荒索要墨宝，这会儿拾荒送我一幅关于小破孩的钢笔墨宝：小破孩和小丫站在一个睡觉的大猪的背上，小破孩好玩地抬起小腿似乎在放一串泡泡屁，小丫在一边惊讶地张开嘴，大猪的身边还立走着一个神气的小猪娃。\
\
拾荒为了让我进一步了解小破孩，让我看一家电视台专访拾荒公司王永和拾荒。\
\
我看了一会儿说：这个片子你可拷入我的移动硬盘。但是我有一个习惯，我写的东西必须是我自己采访来的。您能不能给我讲更多的关于小破孩或是关于你――拾荒的故事。\
\
没想到拾荒这会儿却是牛B起来。拾荒说：我的故事都在小破孩里了。我希望观众关注小破孩而不是关注拾荒。看我一脸迷惑，拾荒解释道：小破孩的许多的故事都有生活来源。关注小破孩就等于关注拾荒。小破孩能说明一切。\
\
我说那你不会觉得老调常弹没有新意？你不是要实现小破孩的产业化吗？\
\
拾荒说许多世界动画名牌，比如“奥斯卡”，迪斯尼的《猫和老鼠》--是他的最爱，比如米老鼠唐老鸭，等的推广也就都是老套常弹，就那点事，反复说反复说，并没有新的东西。\
\
我说：你既然想把小破孩作为一种产业，既然你想做得与众不同，那么你就当为了小破孩适当出卖自己的隐私，你当有一种牺牲精神，否则小破孩的推进速度受限。有说“功夫在诗外”吧。\
\
还有，刚才你让我看采访你的专题片，我发现你动漫做得特别好，但却不是特别能说－－有什么力量从内里把拾荒的幽默感束缚住了一般。内心特别丰富的你并没有把您的丰富感受表达出来。并且你身上小破孩式的幽默并没有释放出来－－这可能是采访记者没能营造出宽松的氛围，也可能是面对陌生环境陌生人你从来就很拘束。在小破孩走向产业化的初始，你是不是也当关注一下自己形象的包装？如果你特别能说能释放出幽默就走到前台，如果你不是特别能说，或是根本不愿说，或是不得不说，或是不愿释放小破孩式幽默，那么你不如干脆就退到幕后，让王永来说，让你哥哥来说。让观众想见你却见不到你。或是见到的是烟雾笼罩的你－－比如是通过剪辑制作出的专题片，在你哥哥和王永介绍到你时播放，或是单独在电视台播放。天长日久，拾荒就如同一个神秘的黑老大，或是一个“酷毕”的黑衣闪客大侠--如同拾荒笔下的中国佐罗，这样更有诱惑力。你形象策划至关重要，因为它和小破孩的成长为产业休戚相关。\
\
拾荒给我介绍了他的哥哥。\
\
拾荒的哥哥田平易和拾荒长得很相，只是个子小一些。兄弟两个都有光亮的小破孩式大脑袋。只是田平易的身上衬衫上有道时髦的折子，少了拾荒的挺拔、俊朗和潇洒，却多出一种落拓不羁。这落拓不羁中又透出一种成熟、实干和执拗。那是一种更加贴近泥土的草根劲儿。田平易是如此自我介绍的：我曾就读于上海师范大学计算机系，毕业后在洛阳某企业销售电话交换机。2002年底，我来上海出差，得知弟弟拾荒创作的“小破孩”系列动画，已经在互联网上名声赫赫。弟弟期望低迷已久的中国动漫业诞生自己的动漫品牌。而我的商业大脑和市场敏感让我意识到作为新兴产业的网络动漫业，正在成为一个爆炸性发展的巨大产业。\
\
2003年，我为弟弟辞职，和弟弟联手投资50万元，成立上海拾荒动画设计有限公司。\
\
我想起拾荒电话中给我透露：刚开始投资只不过2万。从这一点看出兄弟两个的不同：一个是属于商界的，一个是属于艺术界的。\
\
我们将“小破孩”动漫形象进行了注册，正在逐步实现“小破孩”的产化，使“小破孩”走向市场。\
\
2004年，我们聘王永为拾荒公司“首席运营官”，负责该公司的风险投资、战略架构、网络技术。王永是国内资深网络高手，洛阳大学毕业生，23岁时撰写论文就在全国电视广告行业有些影响；1997年就被国际知名软件公司聘为网络专家；1999年中央电视台春节联欢晚会首次实现网络直播，他任直播的技术总监，同年，他主持了河南省第十七届洛阳牡丹花会的网上直播，这也是国内大型经贸活动首次实现网上直播；2000年，他受聘洛阳大学客座教授，建立洛阳大学新媒体实验室和互联网实验室。\
\
三个出类拔萃的洛阳青年的风云际会，我预感将会在中国动漫界演出了一场“动漫大戏”——我预感“小破孩”真能在国内原创动漫形象中率先实现产业化。\
\
拾荒和哥哥田平易邀请我吃饭。在去饭店的路上，田平易说今天晚上吃饭还有两个人，一位是投资公司的郑总，一种是青岛动漫公司的女士某某。\
\
拾荒又在不经意间给我透露商业“机密”。拾荒说：投资公司的人来谈的多，可是谈成合的不多。我问为何，田平易给弟弟“补漏”：投资公司的钱不是好用的。投资公司对我们有很多要求，比如一年要有多少的投资回报率，我们想潜心打造一个中国动漫品牌，可是投资公司只想一年回收多少钱，我们要打动漫品版就不能急功近利，我们的每一步都需要扎扎实实，因这感觉投资公司将给我们施加的压力太大，我们宁肯不用他们给我们的投资。\
\
进了饭店，席上被称作郑总的投资公司经理和一位青岛女士已经在座。我介绍自己是中国石油报记者。田平易在一边补充介绍：刚才毛竹介绍的不全，毛竹不仅是记者还是著名的作家，她的笔名叫东方竹子，竹子。我不知道在这场合，记者和作家有何区别。可能是在拾荒哥俩心里作家比记者更有分量。\
\
我们落座。拾荒哥俩坐在我的两边，我的对面是郑总。斜对面是那位女士。\
\
郑总和我、田平易、青岛女士交谈热烈。我发现拾荒话不多，怕冷落了拾荒，便转身和拾荒交谈。\
\
我说你们哥俩儿长得挺像的，特别是那个小破孩式的大额头大脑袋。我问：你家哥几个。拾荒说：三个，我是老二。拾荒若有所思，又添一句：我家三个儿子，小时候连饭都吃不饱。我说如果吃饱了可能就没有如此有灵气的大男孩拾荒了。因为在饥饿状态人的潜能可得到最大释放。我其实心里一直感谢自己小时候吃不饱。\
\
我说：你和小破孩真的挺神似的，只是你的额头上少了小破孩的“爱情大包”。\
\
我细细地观察拾荒：拾荒的头顶上倒是隐现一个包。这个包使得拾荒的头顶显得有些尖，如同是一个可发射出的原子弹弹头，带有一种突破力和一种创新力，还有一种叛逆者遗世独立孤注一掷的味道。但是拾荒的额头上不仅没有“爱情大包”反而有一点微微凹陷。不知道这是不是“爱情凹陷”。那是神秘的“爱情凹陷”是神秘的“爱情天坑”吗？\
\
我的脑海中闪过浙江境内神秘的天坑，那雾气迷蒙的天坑仿佛真的是接天地灵气的呢！而拾荒的这个“爱情天坑”是不是真是拾荒拽“爱情”时被砸出的？而拾荒的“爱情天坑”是不是用来接天地灵气的？而拾荒特别的灵气是不是都来自这个神秘的“爱情天坑”，不然为何别人创不出中国动漫第一品牌，唯他可以呢？这可真是一个谜呢。\
\
而拾荒让小破孩的额头上有一个包，是不是受头顶上有一个包的启示，只不过是把包从头顶移到额头？是拾荒头上有个“爱情凹陷”想让它鼓起来，故而想出这么一个可爱的“爱情大包”吗？是拾荒受其它启示－－比如寿星老人，画出这么一个“爱情大包”吗？这一点，拾荒不说也不得而知也。反正是拾荒头顶的大包和小破孩额头上那种大包遥相呼应，带出某种神秘的味道。\
\
我问拾荒：是不是有一个小丫做为创作的原型。拾荒说：当然每一个动漫都有许多的故事。我问拾荒：你过而立而没有成家，但你曾经的女友是不是就是一个小丫式人物。拾荒的脸居然微微儿地有些儿红了。我没好再问，如此感觉细腻、灵气聪颖、感情丰富、艺术感觉颇好的大男孩拾荒居然还没有成家，那么拾荒一定是情路坎坷。而他的第一张动漫：梁祝音乐下的小破孩子出世纪，就是送给“天下没有家的你”的。这里面一定有感人的故事。\
\
但是这个场合我不好追问。\
\
而拾荒强调过，小破孩给小丫糖葫芦、牡丹花－－我想当不是玫瑰而是洛阳牡丹，铜钱，月亮，每一样东西都是有深刻含义的。而这些故事一定不仅和拾荒四处飘流，四方求索，最后创办拾荒公司的艰辛分不开，而且和拾荒的爱情故事有关。而拾荒不仅一次地说到洛阳，说到草根，这样重情重义的拾荒身边怎么能没有一个小丫式的草根女友伴它创业呢？这个真是一个谜呢。拾荒的女友是一个小丫式的小店老板吗？拾荒和女友有一系列小丫式的故事吗？不仅是我，也是亿万喜欢小破孩的观众十分想知道的。可是拾荒在回避这类话题，我也不便多问。\
\
拾荒昨天曾对我说：希望您能帮助我分析一下为何这么多的人喜欢小破孩。这种轰轰轰烈烈的对小破孩的喜欢连我都感觉震惊。拾荒很认真地说：看了你的网，感觉里面的文章很有新意。我希望你能试着分析一下。我知道，拾荒是希望我写一篇有关小破孩的论文。\
\
拾荒沉思了一会儿，接着说，我还想知道著名作家怎么看小破孩，比如你认识的贾平凹等名家。因为贾平凹是陕西人，对民间文化很有研究的。我知道他是看了我网上写的“把自己埋在汉陶中幽灵般的贾平凹”。\
\
席上的五人天南地北地聊。郑总说准备在青岛投一个动漫基地，投入四个亿。我心想为何不投在洛阳，因为拾荒不是想把洛阳打造成动漫之都吗？而且有中国特色的成功动漫品牌只有拾荒的小破孩和小丫。\
\
我又发现拾荒思维的游离，似乎是若有所思。我转身问拾荒，你在想什么？拾荒很认真地对我说：点击一下，喂一瓶奶，就会救活王二小，你帮助我推测一下会有多少观众点击喂奶救活王二小？会不会达到一个亿？我一怔，心里又有感动。有说入木三分。有说力透纸背。拾荒是在用他全部的生命无时无刻分分秒秒地琢磨小破孩的创作，沉浸在创作的快乐中。\
\
我说：反正是看了您的《王二小》的人，可能都想给王二小喂这瓶奶去救活王二小。量一定会很大！至于多少我说不准，因为这还和投放的力度、广告的力度等有关。\
\
啥叫“三句话不离本行”。而拾荒真是三句话不离小破孩。拾荒在这种场合下一次一次谈的话题都是小破孩，居然和投资和青岛动漫城都无关，这让我再一次让我感觉到了神奇大男孩拾荒身上绝对的艺术家气质。那是一种种远离商业气氛接近泥土草根的大艺术家气质。更可贵的是拾荒有自知知明，他两次对我说，市场运作这些不是我俩能做的事情，而拾荒给自己的定位也正是全心身的投入创作，不计名不计利不设一切代价地拿出好的原创作品。而拾荒对创作小破孩那种全心身的投入真的让我感动。或许正是因为拾荒这种全部生命的投入，不计一切代价的投入，才让我看到那么好的动漫了呢！才给我们内心花蕾绽放般的惬意，才给我们溪流滚过般的快乐！\
\
而拾荒表现出的非常的潜力耐力毅力定力，还有深沉执拗，都让我感觉到拾荒公司将来一定会有特别的发展。\
\
我说再过五年拾荒公司就会非常的了不起。拾荒哥俩马上说：用不上五年，三年就起来了。我又说：我要是投资公司我会快下手，不然就来不及了，因为这个的品牌中国没有第二个。我的话并不是说给郑总听的，是由衷的，我只是对郑总既然投资动漫城为何不投在洛阳有疑问？而我知道郑总这次来沪洽谈就是准备给小破孩投资的。我仅仅是觉得既然看好投资小破孩，就不要如其它的投资公司给他们三人太大的压力。\
\
第四天，我和张俊玲一起去逛南京路，回路上，张俊玲因为晕机便用二千元的高价包了一节软卧，我们一同回京。我说把这个那个给张俊玲，让她回去送给她的双胞胎女儿，一如小破孩拿出糖葫芦、牡丹花、铜钱、小狗，可是小丫－－张俊玲都没有兴趣，我没有办法，只好拿出小破孩书中几本送给张俊玲的双胞胎女儿。果然张俊玲也是一见就喜欢：这个，我的双胞胎女儿一定会喜欢！－－后回京后张俊玲果然说两个双胞胎女儿喜欢的不得了，问怎么少了两本？我有些不舍得，把光盘和其中两本自己留下。这可是拾荒送给我的珍贵礼物呢！\
\
望着车窗外掠过的长三角地区，我仿佛又看到神奇大男孩拾荒深沉深邃笃定的目光，我一遍一遍地从心里为拾荒公司祝福，祝福拾荒公司尽快实现产业！祝福小破孩更加深入人心！祝福小破孩的成长更加一路顺风！\
\
链接：\
\
最近一次去上海，又一次见到拾荒。拾荒让我在锦江之星宾馆大厅等他接，可是我居然凭记忆顺利找到了藏在锦江之星宾馆后重楼中的小破孩之家。看到拾荒我忍不住感叹：你长得越来越像小破孩了！拾荒告诉我：我们要搬家了！我说搬到哪儿？拾荒说：“到一个更像写字楼的地方。”我忍不住笑了：这不是小破孩的语言又是什么？\
\
我说看到网上采访你，说小破孩艰难到第五年，真的还是那么难吗？拾荒坦率地再次承认：那是，你知道，动漫上亿人看都是免费的！就是艰难生存。我忽然有种安慰，虽然小破孩公司已经走向商业运作，可是小破孩的原创，仍旧是那个不言商的拾荒，那个简单得一如一个小孩子只说实话的拾荒。\
\
我们在锦江餐厅二楼落座，我说小破孩太可爱了，看不够，就是你们的速度太慢了。我听说拾荒压着做好的小破孩动漫不放网站的。拾荒笑，拿出手机来给我他新做的小破孩动漫，是两个关于辑毒的故事。拾荒对我没有看央视播出的《大呆小呆》动漫电视十分惊奇。拾荒拿出手机给我看了《大呆小呆》的主题曲。
